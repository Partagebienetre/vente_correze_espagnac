3. (Option) ajouter les posters story-1.jpg à story-4.jpg dans images/

## Déploiement GitHub Pages
1. Créez un dépôt (ex. terre-et-eau) sur github.com
2. Glissez index.html + les dossiers images/ et video/
3. Settings > Pages > Source : branche main > dossier /(root) > Save
4. Le site sera en ligne sous https://VOTRE-COMPTE.github.io/terre-et-eau/
